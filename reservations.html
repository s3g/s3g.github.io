Reservations

LINK TO RESERVATION SYSTEM (LOGIN REQUIRED)

  • Use of the studio is restricted to students enrolled in hosted courses and the associated faculty and staff.
  • Access to studio and reservation system is granted after an introductory training.
  • After introductory training, it is assumed that users are able to troubleshoot common problems.
  • There is no on-call support for 25.4 loudspeaker array. Submit a support ticket if a problem occurs.
  • Please review the studio rules and guidelines. Failure to adhere to these policies can result in revoked access.
